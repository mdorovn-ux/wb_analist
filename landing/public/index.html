Что внутри

Минимум ручной рутины, максимум сверки

Автоотчет WB

Сводка, детализация, продажи, возвраты, товары и прибыль по товарам в одном интерфейсе.

Точные расходы

Себестоимость товаров, расходы периода, реклама WB, УСН, НДС и ручные налоговые настройки.

WB-периоды

Можно выбирать готовые отчетные периоды WB или задавать диапазон вручную.

Защита от лимитов

Finance API выполняется через очередь, rate limiter и повтор запросов после 429.

Кэш загрузок

Данные сохраняются локально по периоду, типу отчета и кабинету, чтобы не ждать API повторно.

Логи и диагностика

Встроенный раздел логов помогает понять ошибки API, сети и загрузки без передачи токенов.

Как работает

От периода до Excel за несколько шагов

  1. 1

    Добавьте API-ключи

    Finance API обязателен, Promotion API можно подключить для рекламы WB.

  2. 2

    Выберите период

    Используйте WB-периоды для точной сверки или ручной диапазон для быстрой проверки.

  3. 3

    Загрузите отчет

    Программа сама учитывает лимиты API, показывает этапы, строки и прогресс.

  4. 4

    Сохраните Excel

    Получите отчет с вкладками для сверки, прибыли, товаров, рекламы и налогов.
