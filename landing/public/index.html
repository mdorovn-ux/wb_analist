Что внутри

Сверка WB и управленческая прибыль в одном окне

Программа рассчитана на ежедневную работу селлера: выбрать период, загрузить данные, проверить предупреждения и сохранить Excel.

Автоотчет WB

Сводка, детализация, продажи, возвраты, товары, прибыль по товарам и сверка с итогами WB.

Точные расходы

Себестоимость товаров, расходы периода, реклама WB, УСН, НДС и ручные налоговые настройки.

WB-периоды

Можно выбирать готовые отчетные периоды WB или задавать диапазон вручную, если нужен быстрый контроль.

Защита от лимитов

Finance API выполняется через очередь, rate limiter и повторы после временных ошибок или 429.

Локальный кэш

Данные сохраняются по периоду, типу отчета и кабинету, чтобы не смешивать разные API-токены.

Логи и диагностика

Раздел логов помогает разобраться с ошибками API, сети, Excel и предупреждениями без передачи токенов.

Как установить

Скачали ZIP, распаковали, запустили exe

Пользователю не нужно устанавливать Python. Все нужное для запуска находится внутри Windows-сборки.

  1. 1

    Скачайте архив

    Нажмите кнопку скачивания и сохраните ZIP-файл из GitHub Releases.

  2. 2

    Распакуйте папку

    Разархивируйте ZIP в удобное место, например в папку документов или на диск D.

  3. 3

    Запустите программу

    Откройте `WB analyst v1.1.1.exe`. При первом запуске программа попросит ключ активации.

  4. 4

    Добавьте API

    В разделе автоотчета откройте `API Wildberries`, вставьте Finance API token и при необходимости Promotion API token.

Перед скачиванием

Что важно знать

Программа предназначена для Windows и поставляется как ZIP-архив с exe-файлом.

API-ключи пользователь вставляет сам. Токены хранятся локально и не пишутся в логи.

WB Finance API имеет лимиты. Полные отчеты могут загружаться дольше, особенно за большие периоды.

Релизные файлы размещены в GitHub Releases. Пользователь скачивает ZIP без установки Python и дополнительных библиотек.
