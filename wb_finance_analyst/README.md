# WB analyst

Desktop-приложение для финансовой аналитики детализированных отчетов Wildberries. Программа загружает один или несколько сырых Excel-файлов WB, автоматически сопоставляет колонки, считает сверку, продажи, возвраты, логистику, хранение, удержания, штрафы, себестоимость, валовую и чистую прибыль, а затем формирует аккуратный Excel-отчет с техническими листами для последующего объединения.

Текущая версия разработки: `1.1.0-dev`.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Ручная активация

При первом запуске программа показывает код компьютера. Для выдачи ключа запустите локальный калькулятор:

```bash
python tools/license_calculator.py
```

Вставьте код компьютера пользователя, нажмите `Сгенерировать ключ` и отправьте пользователю полученный активационный ключ. Для внутренних тестов в калькуляторе также есть универсальный ключ.

## Сборка EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --windowed --name "WB analyst" main.py
```

## Как пользоваться

1. Откройте раздел `Новый отчет`.
2. Добавьте сырой WB Excel-файл или несколько файлов.
3. При первом запуске проверьте `Настройки колонок` и сохраните сопоставление.
4. Нажмите `Сформировать отчет`.
5. Проверьте карточки сверки, вкладки предпросмотра и предупреждения.
6. Нажмите `Сохранить Excel`.

## Себестоимость

В разделе `Себестоимость и расходы` можно импортировать Excel со столбцами:

```text
Товар
Себестоимость
Упаковка
Комментарий
```

Если товар не найден в справочнике, себестоимость считается равной нулю, а в отчет добавляется предупреждение.

## Объединение отчетов

Раздел `Объединение отчетов` предназначен для Excel-файлов, которые уже сформированы этой программой. Для надежного объединения используются скрытые листы:

```text
_DATA_SUMMARY
_DATA_PRODUCTS
_DATA_OPERATIONS
_DATA_EXPENSES
_DATA_META
```

Итоговые значения `Итого к оплате WB`, `Чистая прибыль` и `Маржинальность %` пересчитываются заново.
