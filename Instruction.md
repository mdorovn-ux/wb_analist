Разработай с нуля полноценную desktop-программу уровня Product для финансовой аналитики отчётов Wildberries.

Название программы:

WB Finance Analyst

Цель программы:

Создать удобное desktop-приложение, которое загружает детализированные Excel-отчёты Wildberries, автоматически распознаёт колонки, считает сверку с WB, продажи, возвраты, удержания, логистику, хранение, штрафы, себестоимость, валовую прибыль, чистую прибыль, формирует подробный Excel-отчёт с несколькими листами и графиками, а также умеет объединять несколько уже сформированных отчётов.

Программа должна быть не тестовой и не MVP, а нормальным продуктовым приложением:

* красивый и аккуратный интерфейс;
* минимум действий со стороны пользователя;
* сохранение настроек;
* обработка ошибок;
* логирование;
* понятные предупреждения;
* аккуратный итоговый Excel;
* возможность объединения отчётов;
* возможность дальнейшего расширения.

# 4. Общая идея программы

Программа должна иметь 4 основные зоны:

```text
1. Новый отчёт
2. Объединение отчётов
3. Себестоимость и расходы
4. Настройки
```

Основной сценарий пользователя:

```text
1. Первый запуск:
   - пользователь открывает настройки колонок;
   - программа автоматически предлагает сопоставления;
   - пользователь проверяет и сохраняет настройки.

2. Дальше:
   - пользователь просто выбирает один или несколько сырых WB Excel-файлов;
   - нажимает "Сформировать отчёт";
   - программа сама считает всё;
   - показывает сверку;
   - показывает предупреждения;
   - сохраняет Excel.

3. При необходимости:
   - пользователь открывает вкладку "Объединение отчётов";
   - добавляет несколько уже сформированных Excel-отчётов;
   - нажимает "Объединить";
   - получает общий отчёт за период.
```

Сформировать таблицу `summary_df` с колонками:

```text
Показатель
Сумма

Комментарий
```

Порядок строк:

```text
Продажа WB
Возвраты WB
Продажа WB чистая
К перечислению по продажам
Возвраты к перечислению
К перечислению за товар
Логистика
Хранение
Операции при приёмке
Удержания/выплаты
Штрафы
Стоимость участия в программе лояльности
Баллы по программе лояльности
Разовое изменение срока перечисления
Корректировка ВВ
Компенсация скидки по программе лояльности
Итого к оплате WB
Себестоимость
Упаковка
Внешние расходы
Чистая прибыль
Маржинальность %
```

Формулы:

```text
Продажа WB =
sum(Вайлдберриз реализовал товар, где Тип документа = Продажа)

Возвраты WB =
sum(Вайлдберриз реализовал товар, где Тип документа = Возврат)

Продажа WB чистая =
Продажа WB - Возвраты WB

К перечислению по продажам =
sum(К перечислению продавцу, где Тип документа = Продажа)

Возвраты к перечислению =
sum(К перечислению продавцу, где Тип документа = Возврат)

К перечислению за товар =
К перечислению по продажам - Возвраты к перечислению
```

Глобальные расходы:

```text
Логистика =
sum(колонка логистики, где Обоснование для оплаты содержит "Логистика")

Хранение =
sum(колонка хранения, где Обоснование для оплаты содержит "Хранение")

Операции при приёмке =
sum(колонка приемки, где Обоснование содержит "приемк" или "приёмк")
если по обоснованию не найдено, использовать сумму всей колонки приемки

Удержания/выплаты =
sum(колонка удержаний, где Обоснование содержит "Удержание")
если по обоснованию не найдено, использовать сумму всей колонки удержаний

Штрафы =
sum(колонка штрафов, где Обоснование содержит "Штраф")
если по обоснованию не найдено, использовать сумму всей колонки штрафов
```

Лояльность:

```text
Стоимость участия в программе лояльности =
sum(колонка loyalty_cost по продажам) - sum(колонка loyalty_cost по возвратам)

Баллы по программе лояльности =
sum(колонка loyalty_points по продажам) - sum(колонка loyalty_points по возвратам)

Компенсация скидки по программе лояльности =
sum(колонка loyalty_compensation по продажам) - sum(колонка loyalty_compensation по возвратам)
```

Корректировки:

```text
Разовое изменение срока перечисления =
sum(соответствующей колонки)

Корректировка ВВ =
sum(соответствующей колонки)
```

Финальная формула:

```text
Итого к оплате WB =
К перечислению за товар
- Логистика
- Хранение
- Операции при приёмке
- Удержания/выплаты
- Штрафы
- Стоимость участия в программе лояльности
- Баллы по программе лояльности
- Разовое изменение срока перечисления
- Корректировка ВВ
+ Компенсация скидки по программе лояльности
```

Важно по знакам:

```text
Если в колонке удержаний или корректировок значение отрицательное, формула должна учитывать знак.
Например:
- (-500) = +500

Не брать abs() без необходимости.
Не превращать все значения в положительные.
```

```text

3. Если технических листов нет, попробовать прочитать обычные листы.
```
